# Projeto-Barbershop

Sistema desenvolvido para automatizar e otimizar a gestão de uma barbearia, proporcionando mais eficiência no atendimento e organização dos agendamentos.

***

## Funcionalidades

- Agendamento de horários

- Cadastro e login de usuários

- Seleção de serviços (corte, barba, etc.)

- Gestão de barbeiros

- Controle de disponibilidade

- Organização dos atendimentos

***

## Objetivo

Facilitar o dia a dia de uma barbearia, reduzindo processos manuais e melhorando a organização dos atendimentos, além de proporcionar uma experiência mais moderna e eficiente para os clientes.

## Linguagens utilizadas

- HTML

- CSS

- MYSQL
